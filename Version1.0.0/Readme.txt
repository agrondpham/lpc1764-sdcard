/*********************************
** 实验说明：
此程序为读取SD卡中的图片,2G的sd卡格式成fat32,把MUSIC文件夹直接拷贝到卡中，然后将卡插到开发板的SD卡槽中（文件夹照片要求，.BMP格式就可以）。
此程序针对SD卡为 FAT32系统不能读图片的缺点进行了修改
** 实验现象：
可以看到图片解码后显示在3.2寸lcd上
